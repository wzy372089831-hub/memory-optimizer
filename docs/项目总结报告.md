## 🎯 核心功能

### 1. 记忆分级 (HOT/WARM/COLD)

- **HOT**: 24 小时内访问，内存缓存
- **WARM**: 7 天内访问，LanceDB 存储
- **COLD**: 超过 7 天，对象存储归档

### 2. 智能搜索

- 向量搜索 (本地模型，无需 API Key)
- 关键词搜索
- 时间/层级过滤
- 混合排序 (相似度 60% + 访问热度 25% + hybrid 15%)

### 3. 定期清理

- 自动归档冷记忆
- 定时删除过期归档
- 支持预览模式 (dry-run)

## 💡 创新点

## 📦 安装与使用

### 安装

```bash
git clone https://github.com/YOUR_USERNAME/memory-optimizer.git
cd memory-optimizer
pip3 install -r requirements.txt
openclaw skill install .
```

### 使用

```bash
# 智能搜索
openclaw skill run memory-opt search "用户偏好" --limit 5

# 查看统计
openclaw skill run memory-opt stats

# 清理冷记忆
openclaw skill run memory-opt cleanup --days 30
```
