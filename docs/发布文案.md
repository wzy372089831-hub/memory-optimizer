## 📖 产品介绍

记忆优化器是 OpenClaw 的记忆优化技能，通过智能分级、向量搜索和定期清理，帮助你：

- 🔧 **配置优化**: LanceDB 索引、缓存、批量操作
- 📊 **记忆分级**: HOT/WARM/COLD 三级管理
- 🔍 **智能搜索**: 向量 + 关键词 + 时间过滤
- ⚡ **按需加载**: 只加载最相关的 3-5 条
- 🧹 **定期清理**: 自动归档冷记忆
- 🛡️ **安全保障**: 回收站、白名单保护、自动备份

## 🚀 快速开始

### 1. 安装

```bash
git clone https://github.com/YOUR_USERNAME/memory-optimizer.git
cd memory-optimizer
pip3 install -r requirements.txt
openclaw skill install .
```

### 2. 配置

编辑 `config.json`:

```json
{
  "lancedb": {
    "path": "~/.openclaw/memory/lancedb",
    "index_type": "IVF_PQ",
    "cache_size_mb": 1024
  },
  "tiering": {
    "hot_threshold_hours": 24,
    "warm_threshold_days": 7
  }
}
```

### 3. 使用

```bash
# 智能搜索
python3 -m src.memory_optimizer search "用户偏好" --limit 5

# 查看统计
python3 -m src.memory_optimizer stats

# 清理冷记忆
python3 -m src.memory_optimizer cleanup --days 30
```

## 🔧 功能特性

### 1. 记忆分级 (HOT/WARM/COLD)

- **HOT**: 24 小时内访问过，优先加载
- **WARM**: 7 天内访问过，按需加载
- **COLD**: 超过 7 天未访问，归档存储

### 2. 智能搜索

- 向量搜索 (本地模型，无需 API Key)
- 关键词搜索
- 时间/层级过滤
- 混合排序 (相似度 + 访问热度)

### 3. 定期清理 (保守策略)

- **90 天**未访问才归档 (宁可少删，不可多删)
- 自动保护重要记忆 (高频/长内容/近期活跃)
- 回收站保留 7 天，可随时恢复
- 支持预览模式 (dry-run)

### 4. 安全保障

- **白名单保护**: protect 命令标记重要记忆永不删除
- **回收站**: trash list/restore 管理已删除记忆
- **自动备份**: 每次清理前自动备份到 ~/.openclaw/memory/backup/
- **二次确认**: 批量删除时要求用户确认
